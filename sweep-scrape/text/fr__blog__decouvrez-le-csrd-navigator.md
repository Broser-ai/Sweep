Votre premier pas vers la conformité CSRD
Si vous ne savez pas par où commencer avec le reporting CSRD, vous n’êtes pas seul. Le processus peut sembler complexe, surtout lorsque l’on entend qu’il faut collecter plus de 1200 points de données. Mais bonne nouvelle : le nombre de données réellement applicables à votre entreprise est bien moins important.
*
Veuillez noter que ces résultats sont basés sur les normes actuelles des ESRS et peuvent être amenés à évoluer après le vote
Omnibus
. Nous mettrons à jour les informations dès que de nouvelles directives seront disponibles.
Méthodologie
Nous avons analysé des centaines d’entreprises de différents secteurs pour vous fournir des informations précises sur le nombre de points de données à collecter, en fonction de votre industrie et de la taille de votre entreprise. Voici les termes que vous trouverez dans le CSRD Navigator :
Données obligatoires
: données que vous devez collecter dès maintenant pour votre reporting.
Données volontaires
: données que vous pouvez choisir d’inclure dans votre reporting.
Données volontaires la première année
: données facultatives la première année, mais qui deviendront obligatoires pour la période de reporting suivante.
Indicateur qualitatif
: vous devrez fournir des données qualitatives (texte).
Indicateur quantitatif
: vous devrez fournir des données quantitatives (chiffres).
Et ensuite ?
Maintenant que vous connaissez le nombre de points de données à déclarer, il est temps de constituer votre liste d’indicateurs et de commencer la collecte de vos données. C’est là que
